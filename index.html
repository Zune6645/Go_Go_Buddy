Introduction

This is a website that lists the locations of amusement parks and playgrounds in the Yangon region. It also includes directions and other special information (such as opening and closing times) for each location.

This project was created in order to enter a competition called WIT (Website Innovation Talent) that was conducted in 2020. This project was created in collaboration with a group, and it was nominated for the finals.
